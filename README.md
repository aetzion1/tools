# adams html tools

Miscellaneous HTML+JavaScript tools built mostly with the help of LLMs. Inspired by [Simon Willison](https://tools.simonwillison.net).

The code lives in [aetzion1/tools](https://github.com/aetzion1/tools) and many tools used Claude custom instructions.

## Miscellaneous
- [Wheel Picker](https://aetzion1.github.io/tools/wheel-picker.html) randomly select text items via interactive wheel
